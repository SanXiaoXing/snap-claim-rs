## v1.0.0
### 功能

- 支持 PDF 文件拖拽/批量添加
- 自动识别电子发票二维码信息
- 支持出行确认单（车票、机票、酒店单）
- 一键生成 Excel 报销单
- 跨平台：Windows / macOS / Linux

### 使用说明

1. 点击「添加 PDF 文件」或拖拽文件导入
2. 选择需要处理的文件（多选支持）
3. 点击「识别」提取发票信息
4. 选择日期范围
5. 点击「导出报告」生成 Excel

### 开发依赖

- Node.js 18+
- Rust 1.70+
- cmake (pdfium 静态编译)
- Visual Studio Build Tools (Windows)

### Python 与 Rust 区别

与之前 Python + Pyside 6对比，Tauri 2的优势在于其运行速度以及其极小的内存占用，能够快速的识别与展示内容。此次项目使用 AI 作为主力改编开发，于此同时进行 Rust 的学习。

### 如何控制屎山代码

由于对Rust的不熟悉，所以很多时候 AI 生成的代码不敢过多删除以及更改，使用 [ponytail skill](https://ponytail.dev/) 进行屎的多拉、堆拉、乱拉等情况。
